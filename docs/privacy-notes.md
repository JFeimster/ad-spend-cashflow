# Privacy Notes

Collect the minimum data needed. Do not collect passwords, bank credentials, marketplace credentials, API keys, or private tokens.

Suggested form language:

> By submitting this form, you agree that your information may be used to generate an educational cash-flow estimate and follow up on your request. Do not submit passwords, bank credentials, private API keys, or sensitive personal information.

If the static calculator only runs locally in the browser, say so. If data is sent to `/api`, disclose that it is processed by your hosted application.
