# Integration Roadmap

## Phase 1: Static launch page

Landing page, on-page calculator preview, Custom GPT CTA, and funding comparison CTA.

## Phase 2: Static-to-GPT bridge

Convert form fields to a GPT-ready prompt and JSON payload with copy buttons.

## Phase 3: Serverless calculator API

Add `/api/forecast-cash-flow`, shared `lib/` modules, and frontend API display.

## Phase 4: Custom GPT Action

Add `actions/openapi.yaml`, connect the GPT Action to Vercel, and test sample requests.

## Phase 5: Lead capture

Add `/api/save-analysis-request`, privacy language, and optional CRM/webhook integration.

## Phase 6: Full app version

Add OpenAI API analysis, structured outputs, saved scenarios, CSV upload, and ecommerce platform integrations only after privacy and security are clear.
