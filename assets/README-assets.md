# Ad Spend Cashflow Image Assets

Add this `assets/images/` folder to the root of your GitHub repo.

## Included files

- `assets/images/banner-1280x640.png`
- `assets/images/banner-1500x600.png`
- `assets/images/banner-1920x1080.png`
- `assets/images/profile-image-1.png`
- `assets/images/profile-image-2.png`

## Suggested usage

Use a wide banner as the hero image or Open Graph preview:

```html
<img src="assets/images/banner-1920x1080.png" alt="Ad Spend Cash Flow Calculator dashboard illustration">
```

Use one of the profile images for cards, favicons, social previews, or app-style visual blocks.
