## Customize

In `index.html`, replace or edit:

- Page title and meta description
- Open Graph image URL
- GPT URL comments if you change the GPT link
- Funding comparison URL if you change the Tally form
- Footer copyright name
- Any copy that needs brand-specific wording

In `styles.css`, edit the `:root` variables to change the theme:

```css
--bg: #070A12;
--surface: #101827;
--text: #F8FAFC;
--muted: #94A3B8;
--primary: #7C3AED;
--secondary: #06B6D4;
--hot: #F97316;
```

In `script.js`, edit the mini calculator logic if your assumptions change.

## Deployment

### Netlify

1. Create a new site in Netlify.
2. Drag the folder containing `index.html`, `styles.css`, `script.js`, and `README.md` into the deploy area.
3. Netlify will publish the static site.

### Vercel

1. Create a new Vercel project.
2. Upload or connect this folder.
3. Use the default static site settings.
4. Deploy.

### GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. Go to **Settings → Pages**.
4. Select the branch and root folder.
5. Save and wait for the GitHub Pages URL.

### Any static host

Upload all files to the same public directory. Keep the file names unchanged unless you also update the links in `index.html`.

## Compliance notes

This page intentionally avoids claiming:

- That the GPT replaces a CFO, CPA, accountant, lender, underwriter, attorney, or tax professional
